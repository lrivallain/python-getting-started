Welcome to a world of Cloud Computing !

This is a sample Python application deployed on Heroku, a cloud computing platform. It's a reasonably simple app - but a good foundation for understanding how to get the most out of the Heroku platform.

Getting Started with Python Source on GitHub

How this sample app works

  • This app was deployed to Heroku, using a git repository.
  • When Heroku received the source code, it fetched all the dependencies in the requirements.txt file, creating a deployable slug.
  • The platform then spins up a dyno, a lightweight container that provides an isolated environment in which the slug can be mounted and executed.
  • You can scale your app, manage it, and deploy over 150 add-on services, from the Dashboard or CLI.
